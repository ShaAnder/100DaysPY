### --- PROJECT: US STATES GAME --- ###

Day 25's project is to create a game in turtle that utilizes csv data reading and enables the user
to guess state names on a map of the US. If the user gets it correct it appends the name to the map.

### --- PARAMETERS --- ###

-The game will start with a blank us map
-The user will click coordinates on the map and enter the state into the prompt
-If correct the state will be filled in if incorrect it will not

-As well as this the game will take note of which states were guessed incorrectly
-At the end it will create a csv called states to learn for the user to learn the states they missed


Provided for the start of the project is a csv with all the states and their rough coords. As well as the blank states img
