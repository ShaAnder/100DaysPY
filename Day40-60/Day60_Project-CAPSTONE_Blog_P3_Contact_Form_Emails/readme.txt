### --- PROJECT NAME / DESC --- ###

The third part of our blog capstone project, the main aim of which is to figure
out how email sending works with flask and fill in html forms

### --- PARAMETERS --- ###

The program should be able to accept information into form fields and then from
there input it into an smtplib module and send it to the user as an email

### --- HOW IT WORKS --- ###

The form data is inputted into the fields and saved as variables
These are then fed to the smtplib module which then sends the email

### --- REQUIREMENTS --- ###

Only real requirement is flask, jinja and knowledge of smtplib
