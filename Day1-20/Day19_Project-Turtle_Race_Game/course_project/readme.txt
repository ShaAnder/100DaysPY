### --- TURTLE RACE PROJECT --- ###

Todays project is going to be a turtle race, using what we know about the coordinate
system and Object state/ instances we are going to make these bois run.

Parameters:

- The program needs 6 turtles
- The user will be given a prompt to bet who wins
- Each starts at the left edge of the screen
- After a brief interval the turtles will race across the screen to the other end
- Each step they take is a random number between 1 - 10
- First turtle to reach the right side wins
- Display a message whether the user won or not
