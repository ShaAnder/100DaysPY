### --- OOP QUIZ --- ###

The project for today is a simple true and false quiz built in OOP the quiz will take
questions from data.py present them to us and we answer, if our answer is correct
gives us a point if not no point. At the end the quiz will tally score and present us our functionality
number. 

With that in mind:

### --- PARAMETERS --- ###

Flow: 

	Greet user -> ask if they want to play 
        -> Program will loop through questions, will ask user a question_model
            -> user inputs an answer, quiz checks the answer key and if it matches awards point
            -> process continues until all questions asked
                -> awards user final score
                -> try again or exit
